# 📊 Представление данных

Веб-приложение на **Streamlit** для полного цикла работы с данными —
загрузка → обработка → виджеты → дашборды. Интерфейс и терминология
приближены к АС «Представление данных» Ситуационного центра: сгруппированное
боковое меню, хлебные крошки, уровни доступа (Общий / Личный), темы оформления.

## 🌐 Онлайн-демо

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://applekh-data-viz.streamlit.app)

Постоянная ссылка: **https://applekh-data-viz.streamlit.app**

## Возможности

| Раздел | Что умеет |
|--------|-----------|
| **Источники данных** | CSV / TSV / Excel / JSON / Parquet, подключение к SQL-базам (PostgreSQL, MySQL, SQLite) и загрузка по REST API / URL |
| **Обработка данных** | Пайплайн преобразований: фильтры, смена типов, переименование, выбор столбцов, удаление пустых/дубликатов, заполнение пропусков, сортировка, группировка с агрегацией, вычисляемые столбцы |
| **Панели → Виджеты** | Гистограмма, линейный график, с областями, точечная, круговая диаграмма, распределение, ящик с усами, список, показатель, процент выполненных работ (gauge), Html, изображение, дата/время. Уровень доступа и тема оформления |
| **Панели → Дашборды** | Сборка виджетов в сетку 1–4 колонки, тема оформления, уровень доступа |

## Установка

```powershell
pip install -r requirements.txt
```

> Драйверы `psycopg2-binary` и `pymysql` нужны только для подключения к
> PostgreSQL / MySQL. Для SQLite ничего дополнительно не требуется.

## Запуск

```powershell
python -m streamlit run app.py
```

Приложение откроется в браузере. Разделы — на сгруппированной панели слева.
Начните с раздела **Источники данных** — загрузите свои данные.

## Развёртывание (постоянная ссылка)

Бесплатно через **Streamlit Community Cloud** — приложение получит постоянный
публичный адрес и будет обновляться при каждом `git push`.

1. Откройте <https://share.streamlit.io> и войдите через **GitHub** (аккаунт `AppleKH`).
2. Нажмите **Create app → Deploy a public app from GitHub**.
3. Заполните:
   - **Repository**: `AppleKH/data-viz`
   - **Branch**: `main`
   - **Main file path**: `app.py`
   - **App URL (Advanced settings)**: задайте поддомен `applekh-data-viz`
     → итоговый адрес `https://applekh-data-viz.streamlit.app`
4. Нажмите **Deploy**. Через 1–2 минуты приложение будет доступно по ссылке.

После деплоя любой `git push` в `main` автоматически обновляет онлайн-версию.

> **Данные на хосте.** Без внешней БД каталог `workspace/` в облаке временный —
> данные сбрасываются при перезапуске контейнера. Чтобы они сохранялись
> постоянно, подключите Postgres — см. раздел **«Постоянное хранилище»** ниже.

## Хранение

Приложение само выбирает бэкенд:

- **Внешняя база (Postgres)** — если задана переменная `DATABASE_URL`
  (или секрет Streamlit `DATABASE_URL`). Датасеты, виджеты и дашборды лежат
  в БД и переживают перезапуск контейнера. **Рекомендуется для облака.**
- **Локальные файлы** (`workspace/`) — если `DATABASE_URL` не задана:
  датасеты — `*.parquet`, виджеты/дашборды — JSON.

Активный режим показан на главной странице.

### Постоянное хранилище (Postgres)

1. Заведите бесплатную базу Postgres — например, **[Neon](https://neon.tech)**
   или **[Supabase](https://supabase.com)**. Скопируйте строку подключения вида
   `postgresql://user:pass@host/db?sslmode=require`.
2. **Локально:** задайте переменную окружения перед запуском —
   ```powershell
   $env:DATABASE_URL = "postgresql://user:pass@host/db?sslmode=require"
   python -m streamlit run app.py
   ```
   или впишите её в `.streamlit/secrets.toml`:
   ```toml
   DATABASE_URL = "postgresql://user:pass@host/db?sslmode=require"
   ```
3. **В облаке (Streamlit Cloud):** Manage app → **Settings → Secrets** →
   добавьте строку `DATABASE_URL = "..."` и сохраните. Приложение перезапустится
   и начнёт хранить данные в БД. Таблицы (`dv_datasets`, `dv_widgets`,
   `dv_dashboards`) создаются автоматически.

> `secrets.toml` не коммитится (он в `.gitignore`) — строка подключения
> остаётся приватной.

## Структура

```
app.py                     точка входа: оформление + сгруппированная навигация
.streamlit/config.toml     настройки Streamlit (тема — через меню Settings)
core/
  ui.py                    фирменный стиль, шапка, хлебные крошки, отрисовка виджетов
  storage.py               хранение датасетов/виджетов/дашбордов
  data_sources.py          загрузка из файлов/SQL/API
  transforms.py            операции обработки данных
  charts.py                построение виджетов (Plotly), типы и темы
views/
  home.py                  главная
  sources.py               источники данных
  transform.py             обработка данных
  widgets.py               панель «Виджеты»
  dashboards.py            панель «Дашборды»
```
